# Dice Simulator
A dice roll simulation written in Python with the Pygame library.

Instructions
===============

<br> - Download and install Python.
<br> - Install Pygame with `pip install pygame`
<br> - Clone this git repository by running `git clone https://github.com/Zeesky-code/Dice-Simulator.git` and navigate to the cloned directory through terminal.
<br> - Run 'python3 dice.py'
<br> - Press Space to roll or stop the dice.
